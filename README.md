# سامانه مدیریت سرویس‌های GPU (پروژه پایان‌ترم)

این پروژه یک وب‌سرویس برای مدیریت و تخصیص منابع GPU است که به عنوان پروژه درس برنامه‌نویسی پیشرفته توسعه داده شده است.

## ویژگی‌ها
- احراز هویت کاربران و ادمین (JWT Auth)
- ثبت و مدیریت درخواست‌های پردازشی (Jobs)
- سیستم صف‌بندی و تغییر وضعیت (Pending -> Approved -> Running)
- شبیه‌سازی اجرای تسک‌ها (بدون نیاز به GPU واقعی)
- رابط کاربری وب (Web UI)

## تکنولوژی‌ها
- **Backend:** Python, FastAPI, SQLAlchemy
- **Database:** SQLite
- **Deployment:** Docker, Docker Compose
